# font attribute 
## schema pattern

See: https://github.com/jsawaya/eigenframe/web/res/font 

Example:
```json
  {
    "type": "TextView",
    "text": "Sub-directories",
    "text_size": "14",
    "text_color": "#ffffff",
    "layout_width": "match_parent",
    "layout_weight": "1",
    "background_color": "#123456",
    "gravity": "center_horizontal",
    "font": "font/open-sans/OpenSans-SemiboldItalic.ttf"
  }
```
