# ToggleButton
## schema pattern

* type = "ToggleButton"
* text_on
* text_off
* checked = true | false (default) 
* id = define unique id to store the view-state
* text_color
* layout_width
* layout_height
* gravity
* background_color
* icon
* padding 
* margin
* on_click action script
